# Portfolio Website - Wireframes & Design Specifications

## 🎨 Design System Overview

### Layout Grid System
```
Desktop: 12-column grid with 24px gutters
Tablet: 8-column grid with 20px gutters  
Mobile: 4-column grid with 16px gutters
```

### Spacing Scale
- **4px** - Micro spacing (between related elements)
- **8px** - Small spacing (between sections)
- **16px** - Base spacing (standard margins)
- **24px** - Medium spacing (section padding)
- **32px** - Large spacing (major sections)
- **48px** - Extra large spacing (hero sections)
- **64px** - Hero spacing (page sections)

## 🎨 Visual Design Specifications

### Typography Hierarchy
```
Hero Heading: 4rem/4.5rem (Bold, 700)
Section Heading: 2.5rem/3rem (Semi-bold, 600)
Subheading: 1.5rem/2rem (Medium, 500)
Body Text: 1rem/1.125rem (Regular, 400)
Small Text: 0.875rem (Regular, 400)
```

### Color Palette
```
Primary: #0f172a (Deep Navy)
Secondary: #3b82f6 (Electric Blue)
Accent: #7c3aed (Rich Purple)
Text Primary: #ffffff (White)
Text Secondary: #64748b (Soft Gray)
Background: #0f172a (Dark) / #fefefe (Light)
```

### Animation Specifications
```
Hover Effects: 0.3s ease-in-out
Page Transitions: 0.5s ease-out
Scroll Animations: 0.8s ease-out
Loading Animations: 1.2s ease-in-out
```

### Interactive Elements
- Smooth scroll navigation
- Hover state animations
- Loading states for forms
- Success/error feedback
- Mobile menu toggle
- Dark/light mode toggle

This comprehensive wireframe system ensures a cohesive, professional, and user-friendly portfolio website that showcases both technical skills and design expertise.
